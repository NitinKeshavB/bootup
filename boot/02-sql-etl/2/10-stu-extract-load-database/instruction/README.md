# Window functions

## Task 

Extract data from the following tables and replicate to your target database. 
- address
- category 
- city 
- country 
- customer 
- film_category 
- film 
- inventory 
- payment 
- rental
- staff 

Make sure to explicitly specify the columns that you wish to extract as these queries serve as a data contract to the data producer.
